# Portfólio Profissional - ADS Uninter

 Este projeto é um portfólio pessoal desenvolvido como parte das atividades da graduação em **Análise e Desenvolvimento de Sistemas (ADS)** no Centro Universitário Internacional UNINTER. O objetivo é demonstrar competências técnicas em Front-end, acessibilidade e boas práticas de desenvolvimento.

## 💾 Tecnologias Utilizadas

* **HTML5:** Estruturação semântica e acessível.
* **CSS3:** Design responsivo, estilização avançada e implementação de **DARK Mode**.
* **JavaScript (Vanilla):** Lógica de troca de temas, manipulação de DOM para conteúdos extras e validação rigorosa de formulários.

## ⚙ Funcionalidades

* **Tema Dinâmico:** Alternância entre Modo Claro e Modo Escuro com preservação de layout.
* **Validação de Formulário:** Sistema que impede envios vazios e valida o formato de e-mail antes do processamento.
* **Interface Interativa:** Botão "Saiba Mais" para exibição de conteúdo adicional sem poluir a interface inicial.
* **Acessibilidade:** Uso de tags semânticas e descrições `alt` em todas as mídias.
